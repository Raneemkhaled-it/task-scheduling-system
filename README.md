# Heap (Priority Queue) Implementation

## Overview

This repository contains my contribution to the IT245 Data Structures group project. My work focused on implementing the Heap (Priority Queue) scheduling algorithm used to process tasks based on their priority.

## My Responsibilities

- Implemented the Priority Queue (Heap) solution.
- Developed the task scheduling algorithm based on task priority.
- Tested and debugged the implementation.
- Analyzed the time complexity of the Heap solution.
- Verified the program output.

## Data Structure Used

- Priority Queue (Heap)

| Operation | Complexity |
